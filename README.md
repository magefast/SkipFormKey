# Magento2 module Strekoza_SkipFormKey
Magento plugin, skip check For Key for chose routes.
Magento module - `Skip validation Form Key`. Can use in Custom solutions/modules

- Add Module name for skip validation Form Key - `\Strekoza\SkipFormKey\Plugin\FormKeyValidatorSkip::ACCEPT_MODULE`
- Base settings for `'checkout', 'liqpay'`



## Install with Composer

`composer require strekoza/module-skip-form-key`
